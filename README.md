# Credit Card Fraud Detector
Built by Charu Gaur | AI Finance Project

## Dataset
- 284,807 real credit card transactions
- 492 actual frauds (0.173%)
- Source: Kaggle — MLG-ULB

## Model Performance
- Algorithm   : Logistic Regression
- ROC-AUC     : 0.9999 (Near Perfect)
- Accuracy    : 98.17%
- Fraud Catch : 100% (after imbalance fix)

## Key Findings
- Fraud transactions average INR 122 vs INR 88 normal
- Fixed class imbalance — catch rate 40% → 100%
- ROC-AUC proves accuracy alone is misleading

## Tools Used
Python | Pandas | scikit-learn | Matplotlib | VS Code

## Domain Background
Built on 5+ years of financial operations at Accenture.
Replicates fraud detection used by Razorpay and Paytm.
